__OVERVIEW__
```
A local desktop AI that understand your instruction, reasons and performs repetitive computer work for you.
```
__PROBLEM STATEMENT__
```
People often do repetitive work and simple tasks, and whether they are in office or at home
they act as laborers, painfully repeating the same work multiple times. There is no simple
desktop application that can handle this work on their behalf, reduce their workload, and
allow them to be more creative and focus on solving bigger problems.
```
